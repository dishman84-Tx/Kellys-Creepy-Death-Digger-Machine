# 🪦 Kelly's Creepy Death Digger Machine v1.3

A high-performance, cinematic desktop application for searching and tracking obituaries across multiple major databases. 

![Reaper Loader](https://raw.githubusercontent.com/jaydeesh579-lang/Kellys-Creepy-Death-Digger-Machine/main/ui/assets/icon.svg)

## 🌟 Features

- **🚀 High-Speed Discovery:** Integrated with the official FamilySearch API for near-instant results.
- **🎬 Cinematic Experience:** Randomized, high-quality MP4 animations (Grim Reapers & Zombies) keep the search engaging.
- **🔍 Deep Search:** Automatically paginates through up to 10 pages of results on Legacy.com and FindAGrave.
- **🧠 Smart Relevance Filter:** AI-grade filtering ensures results match your City, State, and Date Range criteria perfectly.
- **📁 Local Database:** Save your findings to a local SQLite database for offline tracking and historical comparisons.
- **🛑 Advanced Control:** Persistent browser sessions and a "Watchdog" cancel system for smooth, stable operation.

## 📥 Download & Install

To get started, head over to the **[Releases](https://github.com/jaydeesh579-lang/Kellys-Creepy-Death-Digger-Machine/releases)** page and download the latest `KellysCreepyDigger_v1.3_Setup.exe`.

1. Run the installer.
2. Follow the guided setup.
3. Launch from your desktop or start menu.
